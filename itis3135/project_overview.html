Dynamic Functionality with jQuery

This website uses jQuery to provide a smooth and interactive experience for users. Below are the key dynamic features implemented:

  • Booking Form Confirmation: When users submit the booking form, a confirmation message appears showing their selected check-in and check-out dates. Includes cancel/reset functionality.
  • Form Validation: Booking and profile forms check that required fields (like name, email, and dates) are properly filled out before submission.
  • Auto-fill from URL: When a user clicks “Book Now” from a hotel listing, the booking form is automatically pre-filled with the selected hotel name based on a query parameter (e.g., ?hotel=The+Durham+Hotel).
  • Profile Page Interactivity: Users can submit personal info (name, email, phone) and see a success message without reloading the page. jQuery handles the message display and hides it after a few seconds.

All interactive elements were built using jQuery 3.6.0.
